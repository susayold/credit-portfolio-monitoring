# Project 2 Recruiter Summary

Built a governed credit portfolio monitoring and early-warning framework across 1.35M historical booked accounts. The project combines maturity-safe outcomes, portfolio concentration, model drift and calibration, policy trade-offs, proxy ECL stress, synthetic servicing controls, formula-driven Excel, executable SQL/Python, management actions and a seven-view browser dashboard.

The employer value is decision discipline: identify deterioration, isolate its drivers, quantify policy and capacity trade-offs, assign accountable action, and preserve evidence from source through validation. The current public delivery is Web-First Release v1.0 on a reviewer-mode reproducibility checked v2.1.4 analytical baseline. It does not claim live bank implementation, independent validation or organisational approval.
