# Project 2 Web-First Release v1.0

Start with `OPEN_THIS_FIRST.html`.

## Included

- reviewer-mode reproducibility checked analytical baseline v2.1.4;
- governed data, Python, SQL and analytical outputs;
- formula-driven Excel control model;
- risk committee, monitoring and governance reports;
- authoritative interactive browser dashboard;
- UAT, SIT, negative, reconciliation and reproducibility evidence;
- release-specific manifest and clean-extract verification.

## Excluded by design

- Power BI implementation assets and runtime experiments;
- superseded dashboards, screenshots, workbooks and old package manifests;
- local third-party binary caches and temporary runtime workspaces;
- Python cache and compiled files.

These exclusions do not remove governed data, formulas, analytical results or recruiter-facing evidence. Power BI will be handled later as a separate manual implementation plan and is not a completeness gate for this web release.

## Release position

- Browser dashboard: complete and authoritative.
- Excel control model: complete and reconciled.
- Analytical baseline: reviewer-mode reproducibility checked.
- Public claim scope: historical educational monitoring framework.
- Power BI: intentionally outside the release scope.
