# Recruiter Talking Points

## 30-second story

I built a governed monitoring framework for 1.35M historical booked accounts. It connects portfolio quality, model stability, policy trade-offs, proxy ECL, synthetic servicing controls, KRIs and management actions in one browser command center. The value is not just reporting: every breach has a defined interpretation, owner and evidence trail.

## Strongest evidence

- Separates observed, synthetic, proxy and frozen-reference modules.
- Uses maturity-safe outcomes and explicit period eligibility.
- Reconciles Python, SQL, Excel and the browser dashboard.
- Converts monitoring results into risk-appetite status and management action.

## Honest limitation answer

With real bank data I would replace synthetic servicing controls with observed DPD transitions, add applicant-funnel outcomes, validate ECL against accounting policy and obtain independent model-risk review.
