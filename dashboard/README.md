# Project 2 Web Risk Command Center

Open `index.html` directly in a browser.

This is the authoritative recruiter-facing dashboard for Project 2 Web-First Release v1.0. It provides seven interactive views, policy and KRI slicers, evidence drilldown, responsive layouts and explicit observed/synthetic/proxy boundaries.

The live HTML is the public visual evidence. Superseded dashboard screenshots were archived because they no longer represent this interface.
