This project presents a governed credit portfolio monitoring framework that connects portfolio quality, policy outcomes, expected loss, concentration risk, model stability and accountable risk actions.

## Key Results

- 1.348 million accepted/booked accounts monitored.
- 19.42B analytical exposure units assessed.
- 21.35% latest matured bad rate.
- Six red KRIs and one amber KRI translated into named actions.
- 131 automated analytical and control checks passed.

## What The Project Demonstrates

- Current, prior-period and change-based risk monitoring.
- Delinquency, vintage, concentration, policy and expected-loss views.
- Population Stability Index and model-monitoring triggers.
- KRI thresholds, ownership, escalation and action closure.
- SQL marts, formula-driven Excel controls and a seven-view browser dashboard.

## Analytical Boundary

Observed accepted/booked-account evidence is kept separate from synthetic servicing controls. Synthetic monthly delinquency states support implementation and reconciliation testing only; they are not presented as observed bank servicing history.
